## üèÅ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Recommended: Latest LTS)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Execution
Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
